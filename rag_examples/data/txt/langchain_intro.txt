LangChain 是一个用于开发由语言模型驱动的应用程序的框架。
它提供了一套模块化的组件，让开发者可以轻松地构建复杂的 AI 应用。

LangChain 的核心组件包括：
- Models：连接各种大语言模型的接口
- Prompts：管理和优化提示词的工具
- Chains：将多个组件串联起来形成工作流
- Agents：让模型能够自主调用工具和 API
- Memory：管理对话历史和上下文

LangChain 的设计哲学是组合式（Composable）和模块化（Modular）。
开发者可以像搭积木一样，将不同的组件组合起来，构建出复杂的应用。

使用 LangChain 的好处：
1. 标准化接口：统一的 API 调用不同模型
2. 开箱即用：提供大量预构建的组件
3. 灵活扩展：可以轻松添加自定义组件
4. 社区支持：有活跃的社区和丰富的文档

LangChain 适用于以下场景：
- 文档问答系统
- 智能客服机器人
- 数据分析助手
- 研究辅助工具
- 代码生成助手
